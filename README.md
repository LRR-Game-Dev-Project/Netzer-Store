
# 🏪 Netzer Store

## 🎯 מה זה?
משחק סימולציית ניהול זמן תלת־ממדי (3D) לשחקן יחיד: אתם מנהלים את החנות של יצחק — חנות נוחות עמוסה בפיצוחים, ממתקים ומוצרי עישון — וצריכים לשרת לקוחות במהירות, לנהל מלאי ולהתמודד עם הלחץ הגובר ככל שהיום מתקדם.

---

## ⚙️ מאפייני משחק בקצרה
- **פלטפורמה:** מחשב וקונסולות  
- **סגנון גרפי:** פיקסל־ארט/תלת־ממד ריאליסטי  
- **קהל יעד:** כל הגילאים  
- **סוג משחק:** שחקן יחיד (Single Player)

---

## 🎮 רכיבים רשמיים

### 1) שחקנים
- מיועד לכל הגילאים, במיוחד לחובבי משחקי ניהול.  
- משחק לשחקן יחיד — ניהול חנות בזמן אמת, בלי תחרות מול אחרים.  

### 2) יעדים
- **מטרה בכל סשן:** לשרת כמה שיותר לקוחות לפני שיתייאשו.  
- **ניצחון:** הרווחת כמה שיותר כסף ושימור לקוחות מרוצים.  
- **כישלון:** הלקוחות עוזבים → **Game Over**.  
- **הצגה:** מד כסף, מד סבלנות ו־HUD עם בקשות לקוחות.

### 3) תהליכים (Game Loop)
- **התחלה:** מספר לקוחות בודדים ובקשות פשוטות; הדרכה קצרה.  
- **הליבה (Core Loop):**  
  1. קריאת בקשה מלקוח →  
  2. תנועה בין עמדות החנות →  
  3. הכנת ההזמנה →  
  4. קבלת תשלום וניקוד →  
  5. עומס גובר ולקוחות נוספים.  
- **סיום:** עומס יתר או איבוד לקוחות מוביל ל־**Game Over** עם סיכום ניקוד.  

### 4) חוקים
- תגובה איטית או טעות = הלקוח עוזב.  
- פעולה מדויקת ומהירה = כסף וניקוד.  
- בתחילת המשחק מוצגת הדרכה המסבירה את כללי הניהול.  

### 5) משאבים
- **סבלנות הלקוחות:** מוצגת כ־Bar מעל ראשם, יורדת בהדרגה.  
- **מלאי מוצרים:** מאפשר לספק הזמנות; אם נגמר – בזבוז זמן.  
- **כסף:** נצבר ממכירות ומשמש למדידת הצלחה.  
- **איזון:** המשאבים תלויים ישירות ביכולות הניהול של השחקן.  
- **הצגה:** HUD עם ברים ומונה כסף דינמי.  

### 6) עימותים
- **מול המערכת:** עומס לקוחות, זמני תגובה קצרים, מלאי מוגבל.  
- **פנימי:** לחץ ובלבול בזמן אמת — השחקן נדרש לתעדף ולנהל יעילות.  

### 7) גבולות העולם
- **סוג עולם:** חנות סגורה גדולה עם אזורים שונים (פיצוחים, ממתקים, עישון, קופה).  
- **גבולות:** קירות בקצוות המסך.  
- **עקרונות תכנון:** ניידות קלה, התמצאות, עניין והכוונה ברורה בעיצוב הסביבה.

## 🕹️ שליטה
- **תנועה:** מקשי חיצים / WASD.  
- **פעולה:** מקש E או כפתור קונסולה.  
- **ממשק:** HUD פשוט וברור (מד כסף, זמן, וסבלנות לקוחות).
